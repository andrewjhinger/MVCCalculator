#MVCCalculator
###Environment:

MVC web application. Includes Unit Tests.

###Concepts used:

MVC, Unit Testing.

###Functionality:

The page loads to a calculator UI. The user can add, subtract, multiply, and divide. Unit tests are included to validate the logic of the program meets expectations.
